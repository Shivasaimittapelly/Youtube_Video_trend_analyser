
# YouTube Video Trend Analyzer

## Objective
Analyze YouTube trending videos to identify content categories and creators with the highest engagement.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## KPIs
- Views
- Likes
- Comments
- Engagement Rate

## Insights
- Entertainment videos receive the most engagement.
- Popular channels dominate trending videos.
- Views and likes show strong positive correlation.
